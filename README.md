# Журнал Оценок студентов
Данная программа представляет собой простой электронный журнал, в котором можно хранить информацию о студентах и их оценках. Пользователь может добавлять новых студентов, записывать им оценки и просматривать весь список с результатами. Программа работает через текстовое меню и позволяет удобно управлять данными, проверяя правильность ввода и предотвращая ошибки.
